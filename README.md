# Go IoT System

A comprehensive IoT management system consisting of a Go-based backend and a Vue-based frontend dashboard.

## Project Structure

- `go-dashboard/`: Backend API service built with Go, Gin, and GORM.
- `vue-dashboard/`: Frontend dashboard built with Vue 3, Vite, and Tailwind CSS.

## Getting Started

### Prerequisites

- [Go](https://go.dev/doc/install)
- [Node.js](https://nodejs.org/)
- [Docker](https://www.docker.com/) (for MySQL, RabbitMQ, etc.)

### Installation

1. Clone the repository:
   ```bash
   git clone <repository-url>
   cd go-iot
   ```

2. Set up the backend:
   ```bash
   cd go-dashboard
   # Follow instructions in go-dashboard/README.md
   ```

3. Set up the frontend:
   ```bash
   cd vue-dashboard
   # Follow instructions in vue-dashboard/README.md
   ```
